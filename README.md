***Week2-3 Github, fork, pull request***

* Github 필수 명령어   
  * add : 커밋할 목록에 추가
  * commit : 커밋 (히스토리의 한 단위) 만들기
  * push : 현재까지 역사(commit) Github 에 밀어넣기
* fork, pull request
  * fork : 상대방 프로젝트를 복사한다는 의미.
  * pull request : 다른 프로젝트에 내가 만든 commit을 제출한다는 의미. (전송단위 branch)
